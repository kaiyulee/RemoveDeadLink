# 隐私与权限说明（提交到“Privacy practices”）

## 单一用途说明
- 本扩展的唯一目的：检测书签死链并帮助用户移除。

## 数据收集
- 不收集、传输或出售任何个人数据。
- 全部处理在本地浏览器完成。

## 权限用途
### bookmarks
- 用于读取书签树以扫描 URL，并在用户明确操作时移除被判定为死链的书签。
- 不读取或传输出与目的无关的书签信息，所有操作仅在本地进行。

### host permissions（<all_urls>）
- 用于对书签 URL 发起 `HEAD/GET` 请求，验证可达性。
- 不向页面注入内容脚本，不做跟踪或统计；仅在用户发起扫描或删除操作时访问。

### storage
- 使用 `chrome.storage.local` 本地保存用户设置（超时、并发、语言、白名单）与最近一次扫描结果。
- 不同步到任何外部服务；用户可通过移除扩展或清理站点数据来清空。

### 远程代码（Remote code）
- 不下载或执行任何远程代码、脚本或会改变执行逻辑的配置。
- 网络请求仅限：
  - 书签 URL（HEAD/GET）用于连通性校验
  - 公共 RDAP 接口（域名状态/过期）
  - DoH DNS（NXDOMAIN 校验）
- 返回内容不被执行，仅用于判定状态。

## 合规声明
- 遵守 Chrome Web Store 开发者政策：不超范围收集数据、权限用途透明、仅提供明确的单一用途功能。
